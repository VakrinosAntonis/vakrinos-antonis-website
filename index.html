Creating Through Dividing

This toolkit is, modelled in p5.js, explores the potential of Recursive Programming in designing footprints for buildings. The current version focuses on a cathedral standard floorplan, and uses String Physics Simulation with Gravity to replicate Gaudi inspired arches.

Snake Game

A simple implementation of the classic Snake Game using p5.js. The game features a snake that eats food, and the player must navigate it to avoid colliding with itself or the walls. The game includes basic controls for movement and a scoring system to track the player's progress.

GRASSHOPPER DOUBLE CURVED GEOMETRY

Using the Grasshopper extension in Rhino software, this proect models the construction of a double curved geometry. You can find simple python algorithms that feature data tree manipulation and translation to/from vector space.
